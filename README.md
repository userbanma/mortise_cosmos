# 榫合万象 — 平面古建营造交互作品

## 项目简介

《榫合万象》是一款基于 Web 的交互式传统木构建筑营造体验作品。用户通过摄像头手势识别或鼠标操作，在画面中拖拽、拼接榫卯构件，体验中国传统木构建筑的节点工艺，感受"榫卯相合，万象更新"的营造智慧。

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端/启动 | Python 3.10 + 内置 `http.server` |
| 前端框架 | 原生 HTML5 / CSS3 / JavaScript（零框架依赖） |
| 手势识别 | Google MediaPipe Hands（本地离线运行） |
| 图形渲染 | HTML5 Canvas 2D API |
| 数据格式 | JSON（构件配置、彩画数据） |

## 运行方式

### 方式一：Python 启动（推荐）

在项目根目录执行：

```bash
python server.py
```

- 自动启动本地 HTTP 服务器（端口 8081）
- 自动打开浏览器访问项目
- 按 `Ctrl+C` 停止服务器

> 注：MediaPipe 的 WASM 文件需要从 HTTP 协议加载，直接双击打开 `index.html` 无法运行。

### 方式二：其他 HTTP 服务器

也可使用任意静态文件服务器，例如：

```bash
python -m http.server 8081
```

## 功能说明

### 阶段一 · 引导模式（3个子任务）

| 任务 | 内容 | 可移动构件 |
|------|------|-----------|
| 任务一 | 嵌合燕尾榫 × 燕尾卯 | 燕尾榫、燕尾卯 |
| 任务二 | 嵌合直榫 × 直卯 | 直榫、直卯 |
| 任务三 | 将立柱放到已嵌合的梁下方 | 柱 |

- 每个任务仅解锁对应构件，其余构件锁定
- 嵌合成功后有动画反馈和阶段提示
- 三个任务全部完成后出现"自由搭建"入口

### 阶段二 · 自由搭建

点击"自由搭建"后进入自由模式，包含 19 个构件：

| 类型 | 数量 | 规格 |
|------|------|------|
| 直榫 + 直卯 | 4 对 | 小 / 中 / 大 / 特大 |
| 燕尾榫 + 燕尾卯 | 3 对 | 小 / 中 / 大 |
| 柱 | 3 根 | 高 100 / 140 / 180 |
| 屋檐 | 2 个 | 左 / 右 |

- 同类型同尺寸的榫与卯可以配对嵌合
- 所有构件均可自由拖拽、组合、搭建

## 操作说明

| 操作 | 手势模式 | 鼠标模式 |
|------|---------|---------|
| 拖拽构件 | 食指拇指捏合 | 按住左键 |
| 查看构件介绍 | 捏合/点击构件 | 点击构件 |
| 切换模式 | 按 `M` 键 | — |
| 重置 | 点击"重置"按钮 | — |
| 导出插画 | 点击"导出插画"按钮 | — |

## 构件类型

- **直榫 / 直卯**：最基础的榫卯形式，矩形榫头插入矩形凹槽
- **燕尾榫 / 燕尾卯**：梯形榫头，根部窄、端部宽，越拉越紧
- **柱**：垂直承重构件，可吸附到已嵌合的梁组合下方
- **屋檐**：飞檐翘角，用于自由搭建时的屋顶装饰

## 营造值与彩画

- 每成功嵌合一对榫卯，营造值 +1
- 营造值达到阈值后自动解锁传统彩画图案
- 导出插画时可生成 SVG 矢量古建插画

## 开发说明

- 构件形状使用 Canvas 2D API 绘制，包含渐变填充模拟木质厚度感
- 榫卯嵌合动画采用 easeOutBack 缓动，呈现"咔嗒"弹回的嵌合感
- 手势坐标经过 object-fit: cover 裁剪补偿，确保光标与画面精确对齐
- 所有 MediaPipe 库均为本地离线加载，无需联网
