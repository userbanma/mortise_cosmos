# Mortise-Cosmos（榫合万象）

> 平面古建营造交互作品 —— 榫卯相合，万象更新

## 项目简介

《Mortise-Cosmos》是一款基于 Web 的交互式传统木构建筑营造体验作品。用户通过摄像头手势识别或鼠标操作，在画面中拖拽、拼接榫卯构件，体验中国传统木构建筑的节点工艺，感受"榫卯相合，万象更新"的营造智慧。

![项目截图](https://raw.githubusercontent.com/userbanma/mortise-cosmos/main/assets/screenshot.png)

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端/启动 | Python 3.10 + 内置 `http.server` |
| 前端框架 | 原生 HTML5 / CSS3 / JavaScript（零框架依赖） |
| 手势识别 | Google MediaPipe Hands（CDN 加载） |
| 图形渲染 | HTML5 Canvas 2D API |
| 数据格式 | JSON（构件配置、彩画数据） |

## 运行方式

### 方式一：Python 启动（推荐）

在项目根目录执行：

```bash
python server.py
```

- 自动启动本地 HTTP 服务器（端口 8088）
- 自动打开浏览器访问项目
- 按 `Ctrl+C` 停止服务器

> 注：MediaPipe 的 WASM 文件需要从 HTTP 协议加载，直接双击打开 `index.html` 无法运行。

### 方式二：其他 HTTP 服务器

也可使用任意静态文件服务器，例如：

```bash
python -m http.server 8088
```

## 功能说明

### 阶段一 · 引导模式（3 个子任务）

| 任务 | 内容 | 可移动构件 |
|------|------|-----------|
| 任务一 | 嵌合燕尾榫 × 燕尾卯 | 燕尾榫、燕尾卯 |
| 任务二 | 嵌合直榫 × 直卯 | 直榫、直卯 |
| 任务三 | 将立柱放到已嵌合的梁下方 | 柱 |

- 每个任务仅解锁对应构件，其余构件锁定
- 嵌合成功后有动画反馈和阶段提示
- 三个任务全部完成后出现"自由搭建"入口

### 阶段二 · 自由搭建

点击"自由搭建"后进入自由模式，包含多种构件：

| 类型 | 数量 | 规格 |
|------|------|------|
| 直榫 + 直卯 | 4 对 | 小 / 中 / 大 / 特大 |
| 燕尾榫 + 燕尾卯 | 3 对 | 小 / 中 / 大 |
| 柱 | 3 根 | 高 100 / 140 / 180 |
| 屋檐 | 3 个 | 大 / 中 / 小 |
| 雀替 | 6 个 | 左 3 份 / 右 3 份 |

- 同类型同尺寸的榫与卯可以配对嵌合
- 所有构件均可自由拖拽、组合、搭建
- 可将不需要的构件拖入左下角回收区域移除

### 彩画系统

- 每成功嵌合一对榫卯，营造值 +1
- 营造值达到阈值后自动解锁传统彩画图案（和玺彩画、旋子彩画）
- 已解锁的彩画颜料可从左侧托盘拖至已嵌合的构件上染色
- 导出插画时可生成 PNG 格式的古建插画

## 构件类型

- **直榫 / 直卯**：最基础的榫卯形式，矩形榫头插入矩形凹槽
- **燕尾榫 / 燕尾卯**：梯形榫头，根部窄、端部宽，越拉越紧
- **柱**：垂直承重构件，可吸附到已嵌合的梁组合下方
- **屋檐**：飞檐翘角，用于自由搭建时的屋顶装饰（大 / 中 / 小）
- **雀替**：檐下装饰构件，左右成对，龙纹透雕（左 3 份 / 右 3 份）

## 开发说明

### JS 模块拆分

原始 `app.js`（约 2200 行）已拆分为 5 个模块：

| 模块 | 职责 |
|------|------|
| `state.js` | 全局状态 App 定义、彩画图案数据、颜色常量 |
| `components.js` | 构件初始化、碰撞检测、嵌合校验、嵌合动画、构件形状绘制函数 |
| `gesture.js` | MediaPipe Hands 初始化、手势坐标映射、结果回调处理 |
| `rendering.js` | 主渲染循环、彩画托盘绘制、回收区域绘制、光标绘制 |
| `app.js` | 入口：鼠标事件绑定、交互触发器、UI 更新、阶段二逻辑、导出功能 |

### Python 后端拆分

| 文件 | 职责 |
|------|------|
| `config.py` | 服务器配置：端口号、项目目录 |
| `handler.py` | 自定义 HTTP 请求处理器：跨域头、WASM MIME 类型 |
| `server.py` | 主入口：组装配置与处理器，启动服务器并自动打开浏览器 |

### 关键技术点

- 构件形状使用 Canvas 2D API 绘制，部分构件使用图片素材（雀替、屋檐）
- 彩画系统使用低透明度图片纹理叠加在构件上
- 榫卯嵌合动画采用 easeOutBack 缓动，呈现"咔嗒"弹回的嵌合感
- 手势坐标经过 object-fit: cover 裁剪补偿，确保光标与画面精确对齐
- MediaPipe Hands 通过 CDN 加载，减小仓库体积
